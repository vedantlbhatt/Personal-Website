BuzzBus

Android Studio, Firebase, Git

BuzzBus is a web app that aims to help students of the Georgia Institute of Technology navigate campus efficiently. It uses the Haversine formula to find the nearest stops considering Earth's curvature, and an algorithm to compute the fastest routes with one or more bus stops. Its modern interface offers multiple route options, simplifying campus transit without manually exploring maps.

Check it out!

MySQL - Runner

TypeScript, Node.js, Python, Flask API, MySQL, Git, Ollama

mysql-runner is a VS Code extension that lets developers connect directly to their databases and interact with them using natural language. Backed by a Flask API and the Ollama Llama 3.2 model, it translates plain English into SQL queries and executes them to fetch, modify, and alter data, streamlining database management and reducing context switching.
